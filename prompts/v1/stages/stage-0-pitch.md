# Stage 0 Prompt: Initial Pitch

Goal:
Turn a free-form pitch into an initial framing artifact.

You must:

- restate the pitch in clear operational language
- identify what is already clear
- identify what is ambiguous
- identify what is missing
- ask targeted follow-up questions

Blocking concerns include:

- missing project objective
- unclear target users
- unclear domain/problem statement
- contradictory statements in the pitch

The output should prepare the next stage, not prematurely complete it.
