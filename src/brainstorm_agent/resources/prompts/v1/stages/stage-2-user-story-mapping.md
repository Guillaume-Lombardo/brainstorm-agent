# Stage 2 Prompt: User Story Mapping

Goal:
Translate the framed problem into user journeys and capability slices.

You must capture:

- actors and personas
- end-to-end journeys
- capabilities and features
- grouping into MVP, iteration 1, and iteration 2+
- dependencies
- functional gaps
- obvious edge cases

Block transition when:

- journeys are incomplete
- MVP scope is not defensible
- dependencies are unclear
- key user capabilities are missing or contradictory
