# Stage 1 Prompt: Problem Framing and 5W1H

Goal:
Produce a rigorous problem framing with explicit 5W1H coverage.

You must capture:

- business problem
- users and actors
- measurable objectives
- constraints
- explicit non-goals
- assumptions
- initial risks
- who, what, when, where, why, and how

Block transition when any of the following remains too vague:

- problem statement
- target users or actors
- measurable success criteria
- material constraints
- non-goals

Challenge hand-wavy goals, generic personas, and missing constraints.
